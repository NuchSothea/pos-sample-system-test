# POS Sample System Test

This repository contains a sample Point of Sale (POS) system
along with unit, integration, and system test cases.

## Features
- Sales processing
- Tax calculation
- Payment handling (cash/card mock)
- Receipt generation

## Test Types
- Unit Tests
- Integration Tests
- System / End-to-End Tests

## How to Run Tests
```bash
pip install -r requirements.txt
pytest
